# Design Context — Solana Score by Slice Analytics

## Target Audience
Solana wallet holders (crypto-native, 18-35), protocol teams, and AI agents. Users are checking their on-chain reputation score (0-100). They want to flex scores, compare with others, and understand their wallet's strengths across DeFi sectors.

## Use Cases
- Look up any wallet's score and tier
- See breakdown: Volume, Protocol Diversity, Activity, Holdings
- See sector scores: DEX, Launchpad, Aggregator, Perps, Trading Tools
- Share scorecard on X/Twitter
- Protocols query scores via paid API (x402)

## Brand Personality
Crypto-native but approachable. Data-dense but not boring. The team wants to break away from the "clean AI slop" that every web3 dashboard looks like. We're exploring nostalgic 90s/2000s UI themes that tie into the scoring/sybil detection narrative.

## Current Stack
Next.js 16, TypeScript, Tailwind CSS, shadcn/ui, dark-only theme

## Current Colors
- Background: #0c0b14
- Purple: #9945FF
- Green: #14F195
- Cyan: #00D1FF
- Glass cards: rgba(255,255,255,0.035) with backdrop-blur

## Data Points
- Score (0-100), 7 tiers (Whale → Sybil)
- 4 breakdown bars: Volume, Protocol Diversity, Activity, Holdings
- 5 sector scores: DEX/Spot, Launchpad, Aggregator, Perpetuals, Trading Tools
- Stat boxes: Sybil Check (Pass/Fail), Network Fees, Wallet Age, Protocols Used
- Leaderboard, score distribution, protocol grid
